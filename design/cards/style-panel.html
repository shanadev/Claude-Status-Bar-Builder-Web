Behavior

partsicon / label / value each carry independent fg+bg; part selector scopes the swatches and picker below it
swatchesfirst row = theme palette (the 7 neons + surfaces); picker + hex for truecolor; 16-ANSI and 256 modes swap the swatch set
live edithex field accepts paste; picker drag updates the terminal at 60fps (theme write debounced to idle)
thresholdsramp editor appears only for numeric elements; stops drag horizontally; colors come from the palette
empty stateno selection → panel shows theme-level controls: separator style (chevron/round/slant/flame/plain), gap, capsule vs flat, palette preset
resetevery section label gets a "↺" on hover — reverts to theme default
