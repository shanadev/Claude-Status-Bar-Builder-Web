Art direction — v1

Hack the Claude Status Bar

Full Gibson. Hackers (1995) by way of a modern design system — neon on black glass, CRT phosphor, mono type everywhere. Slick, not kitsch. CSS-only effects, fully responsive, prefers-reduced-motion respected throughout.

Pillars

The terminal is the heroThe live xterm.js preview is center stage. Chrome recedes: dark panels, hairline borders, neon only where it earns attention.
The Gibson, not the garbage filePremium neon minimalism. One or two glowing accents per view, generous darkness between them. Restraint is the aesthetic.
CRT authenticityScanlines, phosphor glow, blinking block cursor, glitch on transition. Subtle enough to work for eight hours straight.
Zero-weight effectsEverything is CSS: gradients, clip-path glitch, transform grids. No canvas backdrops, no video, no external fonts beyond the site's own.

Voice

Hacker-movie flavor in small doses: button copy and empty states may say jack in, crash and burn, mess with the best. Instructional copy stays plain and precise. Never let a joke block comprehension of an install step.

Rules

Always
  • Mono type for every glyph on the site
  • Neon = interactive or semantic; gray = inert
  • Static fallback for every animation
  • AA contrast for body text on all surfaces
Never
  • More than two neon hues glowing in one region
  • Glitch effects on text the user is reading
  • Motion that loops faster than 3s in the user's focus area
  • Easter eggs that gate or obscure real functionality
